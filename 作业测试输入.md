# 作业人工测试输入

## 测试目标

测试流程：

1. 教师布置编程作业。
2. 学生提交不同类型的错误代码。
3. 观察系统是否能通过测试用例、AI 复核或薄弱点分析捕捉到问题。
4. 最后提交正确答案，观察作业状态和薄弱点变化。

## 教师布置作业

作业标题：

```text
数组最大值与循环边界测试
```

题目标题：

```text
求数组最大值
```

题目描述：

```text
从标准输入读取一个整数 n，随后读取 n 个整数，输出这 n 个整数中的最大值。

输入格式：
第一行一个整数 n
第二行 n 个整数

输出格式：
输出最大值

要求：
使用 Java 编写，主类名必须为 Main。
```

AI 评分关注点：

```text
循环边界、数组下标、负数处理、输入读取、初始值设置
```

## 学生错误提交 1：循环边界错误

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int[] arr = new int[n];

        for (int i = 0; i <= n; i++) {
            arr[i] = sc.nextInt();
        }

        int max = arr[0];
        for (int i = 1; i < n; i++) {
            if (arr[i] > max) {
                max = arr[i];
            }
        }

        System.out.println(max);
    }
}
```

预期能捕捉到：

```text
数组下标越界
循环边界
for 循环条件
```

## 学生错误提交 2：最大值初始值错误

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int max = 0;

        for (int i = 0; i < n; i++) {
            int x = sc.nextInt();
            if (x > max) {
                max = x;
            }
        }

        System.out.println(max);
    }
}
```

这个提交会在全负数测试用例中失败。

预期能捕捉到：

```text
初始值设置
边界情况
负数处理
最大值算法
```

## 正确答案对照

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int max = sc.nextInt();

        for (int i = 1; i < n; i++) {
            int x = sc.nextInt();
            if (x > max) {
                max = x;
            }
        }

        System.out.println(max);
    }
}
```

## 建议测试顺序

```text
1. 先提交错误 2：观察系统是否识别初始值和负数处理问题。
2. 再提交错误 1：观察系统是否识别数组越界和循环边界问题。
3. 再提交错误 4：观察系统是否识别比较逻辑问题。
4. 再提交错误 5：观察系统是否识别语法或 API 调用问题。
5. 最后提交正确答案：观察作业状态是否通过，薄弱点是否有变化。
```
